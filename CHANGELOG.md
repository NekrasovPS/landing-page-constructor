# Changelog

## [Не выпущено]

### Добавлено
- Горячие клавиши (Ctrl+S, Ctrl+Z, Ctrl+Y, Delete, Ctrl+D, Ctrl+P)
- Тосты-уведомления вместо alert()
- Дублирование блоков (Ctrl+D)
- Мини-превью для перетаскиваемых блоков
- 4 новых блока (Gallery, FAQ, Team, Steps)
- Модальное окно предпросмотра
- Футуристичный UI/UX дизайн
- Слои (Layers Panel)
- Undo/Redo (50 действий)
- Автосохранение в localStorage
- Экспорт/Импорт проектов (JSON + ZIP)
- Шаблоны лендингов (3 готовых)
- Предпросмотр на устройствах (Desktop/Tablet/Mobile)

### Изменено
- Все названия блоков переведены на русский
- Улучшен дизайн EditPanel
- Обновлен Header с градиентами
- BlockVariantsSlider теперь перекрывает панель
- Улучшена анимация перетаскивания

### Исправлено
- Дрожание блоков при сортировке
- Конфликт drag-and-drop обработчиков
- Выход текста за границы при перетаскивании
